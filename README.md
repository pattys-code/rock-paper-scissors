# Rock, Paper, Scissors 

Mini game for "Rock, Paper, scissors". This exercise is part of the "Odin Project" curriculum. 

*How to play:* You can acces the game by opening the latest GitHub Pages deplyoment. Then simply choose an option and click the button below it. The game can be played until either the computer or the human player has 5 points. The score is kept and the winner announced at the end. Have fun playing!
